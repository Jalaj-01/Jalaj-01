#### 04 / OptiSeats `Utility Tool`
> **Algorithmic Exam Seating Allocation System** — *[optiseat.streamlit.app](https://optiseat.streamlit.app/)*
>
> An automated exam room scheduler that maps exam seat distributions, strictly satisfying proximity and branch separation constraints.
